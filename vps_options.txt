## Quick VPS Setup:
```bash
# After SSH to VPS:
sudo apt update
sudo apt install python3-pip git screen
git clone https://github.com/Fatttttr/hsjsn.git
cd hsjsn
git checkout bot-vpn-checker
pip3 install -r bot_requirements.txt

# Edit config with credentials
nano bot_config.json

# Run with screen for persistent:
screen -S vpn-bot
python3 bot_vpn_checker.py
# Ctrl+A, D to detach

# Check later with:
screen -r vpn-bot
```
